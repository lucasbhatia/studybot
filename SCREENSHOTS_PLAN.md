## How to Create Screenshots Manually

### Method 1: Using Chrome DevTools
1. Load extension in dev mode
2. Open sidepanel to desired state
3. Resize window to 1280x800
4. Open DevTools (F12)
5. Take screenshot using DevTools camera icon
6. Crop if needed

### Method 2: Using macOS Screenshot Tool
1. Cmd+Shift+5 (macOS screenshot tool)
2. Select "Capture entire screen" or region
3. Save as PNG
4. Use Preview.app to crop to 1280x800

### Method 3: Using Browser Extension
1. Install "Screenshot" Chrome extension
2. Click extension, select full page screenshot
3. Crop to desired size

## Tips for Great Screenshots

1. **Use real content:** Don't use placeholder text
2. **Show completed actions:** Show a study set with data, not empty states
3. **Highlight key features:** Make interactive elements obvious
4. **Use consistent branding:** Match the StudyBot color scheme
5. **Add context:** Text overlays help users understand what they're seeing
6. **Test on different screens:** Ensure readability on mobile and desktop
7. **Keep it clean:** Remove browser chrome if possible (use DevTools)
